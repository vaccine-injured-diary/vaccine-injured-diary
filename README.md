現役トラッカー太星の
コロナワクチン後遺症治療日記！！
（皮膚炎・自己免疫疾患）

著者　現役大型トラッカー親父　太星

はじめに

読者の皆さん、はじめまして。
オイラは、取り立てて何の特技もない、頭も容姿も良くない、ただの還暦まじかのおっさんです。
自慢できるもん・・・なんかあるかな？　　あった！　　不運の多さには自信があります！！

何故か年に数回、命の危機、みたいなもんがあって、危機一髪の事故が、しょっちゅうあります。
不幸ヒット通算何度目だろうか、ワクチン打ってから、おなじみの大ピンチを迎えました。
でも、こういうの慣れっこ。絶対満塁ホームランで返しちゃる！

ワクチン後遺症にも、すぐに良くなった人だけじゃなくて、もう取り返しのつかない状態で、
人生を諦めている方もいらっしゃると思います。
この本は、俺の闘病記録です。たくさん、このワクチンで苦しんでいる人がいると思うから、
なんらか参考になればと思い、この本を書きました。

ワクチンを打ってから、周りで明らかに何かが変わった。
俺の親戚だけでも、帯状疱疹になる人が、やたらと増えました。
ガンになった人も増えた気がする。流産の話も沢山聞く。
腕が上がらなくなったとか、歩けなくなったとか。
鎮静剤飲むと痙攣するようになったとか、ともかくあのワクチン以来、わけのわからない病が増えました。

公式には一切認められていない。俺の被害妄想かもしれない。でも、公式な調査は一切行われない。
更に、後遺症自体が「臭いモノにフタ」「なかったこと」にされようとしている。
でも、今世界中で、あのmRNAワクチンを打った後、世界中の人の体の何かが変わった。いや、変えられてしまった・・・。

ワクチン後遺症の症状は人それぞれだけど、多くの人が訴えているのは、長引く倦怠感、疲労感、皮膚炎、頭痛、
関節や筋肉の痛み、心臓の問題。
俺の場合は、気の狂いそうなほど痒い、どす黒く変化した皮膚炎と自己免疫疾患だった。
一時期は、体中から血が噴き出てきて、人生諦める覚悟もしてました。
長きに渡る皮膚炎、不快な症状、まだ依然として状況は予断を許さない。

この病気は、医療の現場でも認知が進んでいないし、検査すらできない施設が多いでしょう。また、制度的な問題もある。
いくつもの病院を巡りましたが、ワクチンが原因では？って言うと、俺の会ってきた医師たちは、なぜか非協力的で、なんの検査もしようとしませんでした。怒り出す医師もいました。
でも、ワクチン後遺症の患者を、命がけで救おうとしてくださっている慈悲と気概を持つ、本物の医者の方々もいらっしゃいます。

多分認めてしまうと、戦後最大の薬害事件となってしまう事。病院での自分の立場が危うくなる事。
そして、接種率は全国民の八割ですから、賠償金だけでも、天文学的な数字になるでしょう。
また政治家の逮捕も続出してしまうでしょうから、いつものように、臭いものにフタを被せるしかありません。
僕らの病状の記録でさえ、今後消去される可能性があります。未だ苦しんでいる貴方を置いて。そして奮闘している先生方を忘れて。
こういった手際の良さだけは、いつも感心しざるを得ません。

そうです、もしこのワクチンが薬害だと認めてしまえば、国を根底から揺るがしかねない事態になるからです。
結局被害者たちの事は、どうでもいいのです。
一部の医師は、残念ながら、自分の保身の事ばかり考えているはずです。
でも、本当は、苦しみ続けている被害者の方々、
そして大切な人を亡くされた遺族の方々の気持ちを考えるべきかと思います。

多分このままでは、ワクチン後遺症の全てが、闇の中に葬られる事でしょう。
そして、苦しみ続ける人は、ずっと・・・、死ぬまで、いや死んでも永遠に苦しみ続ける。
だからこそ、自分の命を賭けて、頑張ってくださっている医師や学者の方々、そして被害者たちが、自分たちでどうにかするしかない。

確かに、いったん壊れたものは、完全に元に戻すことは難しい。
でも、俺は諦めない。諦めてたまるもんか！
それは、俺の体だけじゃない、貴方の体も、俺は諦めていない！！

読者の中には、俺と違って、そもそも戦えない方もいらっしゃると思います。
だから、常に本の向こうにいる貴方に変わって、俺は今日も、この病気に挑戦してます。
常識はずれなハードな事にも挑戦してきました。
ですので、このまま真似だけはしないでください。かなり危ない事もしてます。これは絶対の約束です。

助けのないこの病気、私のこの拙い文章だけでも、苦しむ皆さんの少しでもお役に立てるよう祈って書いてます。

柄にもなく、夜空を見上げるたびに、あなたのことを流れ星に願います。
俺は頑張ります。自分のことも大事にしながら、あなたの体を一刻も早く元に戻したい。
少し脱線するかもしれませんが、この本は、日々思っていることや、俺自身を実験サンプルとして記録を書き記したものです。

この本が、あなたの剣となり、盾となり、あなたの体と心を護りますよう。
頑張ってくださっている先生方の資料になれますように。
日本と、あなたの上に垂れ込める暗黒の雲を薙ぎ払い、明日を切り開く破邪の剣になりますよう。
